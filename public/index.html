SSC

Welcome to SSC

Lets take a quick sneak peak at what you can do!

Simple Site Generation

SSC aims to be a simple, static site compiler. Getting up and running takes minutes, letting you not worry about the setup and get right to making your website!

Dynamic Imports

Load in partials, layouts, and more, with powerful directives.

Starter Project with Bootstrap

Get started quickly with a preconfigured project using bootstrap.

"Serverside" Scripting in Javascript

You have full access to the full site and current page objects via javascript within each page, partial, and layout. Utilize powerful functional methods with UnderscoreJS while writing server side javascript.

Meta Data

Data about pages, layouts, partials, all intelligently flow through the site.

Templating

Utilize Markdown and MustacheJS to accomplish all the templeting you need!
